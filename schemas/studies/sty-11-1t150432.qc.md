## Study name says Populus trichocarpa; data says Populus deltoides

The bermap study description says Populus root and rhizosphere from Tennessee, generic. But sample names explicitly identify the host as **Populus deltoides** (eastern cottonwood), not P. trichocarpa:

```
Endosphere MetaG P. deltoides DD176-2
Endosphere MetaG P. deltoides DD176-4
Endosphere MetaG P. deltoides DD176-3
```

`DD176` is the host genotype ID. This is consistent with a Tennessee site (P. deltoides is native to the southeastern US; P. trichocarpa is Pacific Northwest). Worth flagging — bermap study description is generic; the data is P. deltoides specific.

## Open items

- Three sampling compartments (endosphere, rhizoplane, rhizosphere) are correctly distinguished in env_triad — good NMDC curation

- "DD" prefix vs `sty-11-r2h77870`'s `BESC` and `GW` prefixes — different genotype collection conventions, presumably different breeding programs
